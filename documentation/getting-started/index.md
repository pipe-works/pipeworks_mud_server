# Getting Started

Welcome to The Undertaking! This section will help you get up and running quickly.

## Prerequisites

Before you begin, make sure you have:

- Python 3.12 or higher
- pip (Python package manager)
- Git (for cloning the repository)
- A terminal/command line interface
- A web browser

## What You'll Learn

In this section, you'll learn how to:

1. **Install the server** - Set up the development environment and install dependencies
2. **Initialize the database** - Create the SQLite database and default user
3. **Start the server** - Run both the backend API and frontend client
4. **Access the game** - Connect via your web browser
5. **Explore the world** - Navigate rooms, pick up items, and chat with others
6. **Understand commands** - Learn the available commands and how to use them
